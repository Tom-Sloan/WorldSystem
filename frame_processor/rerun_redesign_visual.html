# Enhanced Rerun Blueprint Configuration for 0.23.2
import rerun as rr
import rerun.blueprint as bp

def create_enhanced_blueprint():
    blueprint = bp.Blueprint(
        bp.Horizontal(
            # Left: Live monitoring components
            bp.Vertical(
                bp.SpaceView(
                    name="üìπ Live Camera View",
                    origin="/live/camera",
                    contents=[
                        "/live/camera/**",
                        "/live/detections/**",
                        "/live/tracking/**"
                    ]
                ),
                bp.Horizontal(
                    bp.TextDocumentView(
                        name="üéØ Selected Object",
                        origin="/live/selected_object"
                    ),
                    bp.TimeSeriesView(
                        name="‚è±Ô∏è Timeline",
                        origin="/live/timeline"
                    ),
                    column_shares=[1, 1]
                ),
                row_shares=[3, 1]
            ),
            # Right: Gallery and statistics
            bp.Vertical(
                bp.SpaceView(
                    name="üñºÔ∏è Gallery",
                    origin="/gallery",
                    contents=["/gallery/**"]
                ),
                bp.Horizontal(
                    bp.TextDocumentView(
                        name="üìã Results",
                        origin="/gallery/results"
                    ),
                    bp.TextDocumentView(
                        name="üìä Statistics",
                        origin="/stats"
                    ),
                    column_shares=[2, 1]
                ),
                row_shares=[2, 1]
            ),
            column_shares=[3, 2]
        )
    )
    return blueprint

Key Features of This Design

  • ‚úì
    Dual-View Layout: Both live monitoring and gallery are visible simultaneously - no tabs needed
  • ‚úì
    Flexible Viewing: Use Rerun's entity tree to collapse/expand sections as needed
  • ‚úì
    Integrated Overlays: Bounding boxes with labels appear directly on the camera feed
  • ‚úì
    Real-time Gallery: Processed objects appear immediately in the gallery view
  • ‚úì
    Comprehensive Statistics: Live performance metrics and scene scale in one place
  • ‚úì
    Path Organization: Clear separation between /live/* and /gallery/* paths for easy filtering

How to Use in Rerun Viewer

To focus on live monitoring:

  • Click the gallery panel header and drag to minimize
  • Or collapse /gallery/* in the entity tree

To focus on processed results:

  • Maximize the gallery panel by dragging
  • Or collapse /live/* paths in the entity tree

To see specific data:

  • Use the entity tree checkboxes to show/hide specific paths
  • Drag panel dividers to adjust sizes
  • Double-click panel headers to maximize/restore
